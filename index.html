    List Of Events

    Robo Soccer

    RoboSoccer is a competitive robotics event where teams build robots to play soccer inside an arena. The objective is to score goals by controlling, passing, and defending effectively. This event tests robot control, teamwork, strategy, speed, and precision, delivering intense matches and an exciting experience for both participants and spectators.

    Robo Sumo

    RoboSumo is a power driven robotics competition where robots battle inside a circular ring. The goal is to push or disable the opponent robot. This event focuses on strength, balance, traction, control, and smart design, creating thrilling encounters that showcase mechanical skill, strategic thinking, and competitive spirit among participants worldwide.

    RC Racing

    RC Racing is a high speed robotic race where participants control robots through a challenging track. The aim is to finish the course in minimum time. This event evaluates speed, stability, control, design efficiency, and driving skills, offering an action packed competition filled with excitement, precision, and adrenaline during races.

    Line Follower

    Line Follower is a robotics challenge where robots autonomously follow a predefined path accurately. The robot must detect and track the line using sensors efficiently. This event tests sensor integration, control algorithms, speed optimization, and consistency, rewarding robots that complete the track successfully with accuracy, stability, and minimal deviation errors.

    Robo Run

    Robo Run is an obstacle based robotics race where robots navigate through multiple challenging hurdles. The objective is to complete the course in the least time. This event examines mobility, durability, control, and problem solving skills, rewarding robots that finish fastest while maintaining stability, accuracy, and performance under competitive conditions.

    GameVerse

    GameVerse is a non technical entertainment event featuring multiple fun based games. Participants compete in engaging activities designed purely for enjoyment and excitement. This event offers attractive prize money, encouraging teamwork, creativity, and participation, while providing a relaxed atmosphere where players enjoy games, laughter, rewards, and memorable experiences together happily.
